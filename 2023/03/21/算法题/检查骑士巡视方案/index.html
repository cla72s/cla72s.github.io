检查骑士巡视方案

发表于 2023-03-21 更新于 2023-03-25 分类于 算法题

骑士在一张 n x n 的棋盘上巡视。在有效的巡视方案中，骑士会从棋盘的 左上角 出发，并且访问棋盘上的每个格子 恰好一次 。

给你一个 n x n 的整数矩阵 grid ，由范围 [0, n * n - 1] 内的不同整数组成，其中 grid[row][col] 表示单元格 (row, col) 是骑士访问的第 grid[row][col] 个单元格。骑士的行动是从下标 0 开始的。

如果 grid 表示了骑士的有效巡视方案，返回 true；否则返回 false。

注意，骑士行动时可以垂直移动两个格子且水平移动一个格子，或水平移动两个格子且垂直移动一个格子。下图展示了骑士从某个格子出发可能的八种行动路线。

示例 1：

输入：grid = [[0,11,16,5,20],[17,4,19,10,15],[12,1,8,21,6],[3,18,23,14,9],[24,13,2,7,22]]
输出：true
解释：grid 如上图所示，可以证明这是一个有效的巡视方案。

示例 2：

输入：grid = [[0,3,6],[5,8,1],[2,7,4]]
输出：false
解释：grid 如上图所示，考虑到骑士第 7 次行动后的位置，第 8 次行动是无效的。

提示：

  • n == grid.length == grid[i].length
  • 3 <= n <= 7
  • 0 <= grid[row][col] < n * n
  • grid 中的所有整数 互不相同

解题一：深度遍历搜索

class Solution {
public:
    vector<vector<int>> k{{1,2},{2,1},{-1,-2},{-2,-1},{-1,2},{-2,1},{1,-2},{2,-1}};
    bool move(vector<vector<int>>& grid,int & x,int & y,int i){
        int n = grid.size();
        for (int j = 0;j < 8;++j){
            if (x + k[j][0] >= 0 && x + k[j][0] < n && y + k[j][1] >= 0 && y + k[j][1] < n && grid[x + k[j][0]][y + k[j][1]] == i){
                x += k[j][0];
                y += k[j][1];
                return true;
            }
        }
        return false;
    }
    bool checkValidGrid(vector<vector<int>>& grid) {
        if (grid[0][0] != 0) return false;
        int n = grid.size(), x = 0, y = 0;
        for (int i = 1;i < n * n;++i){
            if (!move(grid,x,y,i)){
                return false;
            }
        }
        return true;
    }
};

复杂度分析：

  • 时间复杂度：O(n^2)
  • 空间复杂度：O（1)

解题二：哈希表

class Solution {
public:
    bool checkValidGrid(vector<vector<int>>& grid) {
        if (grid[0][0] != 0) return false;
        unordered_map <int,int> hash;
        int n = grid.size();

        for (int i = 0;i < n;++i){
            for(int j = 0;j < n;++j){
                hash[grid[i][j]] = i * n + j;
            }
        }

        for (int i = 1;i < n*n;++i){
            int dx1 = hash[i] % n, dy1 = hash[i] / n,
                dx2 = hash[i - 1] % n, dy2 = hash[i - 1] / n;
            if (!((abs(dx1 - dx2) == 1 && abs(dy1 - dy2) == 2) || 
            (abs(dx1 - dx2) == 2 && abs(dy1 - dy2) == 1)))
                return false;
        }
        return true;
    }
};

复杂度分析：

  • 时间复杂度：O(n^2)
  • 空间复杂度：O(n)

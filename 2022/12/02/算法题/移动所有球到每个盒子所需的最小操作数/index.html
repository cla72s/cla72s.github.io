移动所有球到每个盒子所需的最小操作数

发表于 2022-12-02 分类于 算法题

有 n 个盒子。给你一个长度为 n 的二进制字符串 boxes ，其中 boxes[i] 的值为 '0' 表示第 i 个盒子是 空 的，而 boxes[i] 的值为 '1' 表示盒子里有 一个 小球。

在一步操作中，你可以将 一个 小球从某个盒子移动到一个与之相邻的盒子中。

返回一个长度为 n 的数组 answer ，其中 answer[i] 是将所有小球移动到第 i 个盒子所需的 最小 操作数。

每个 answer[i] 都需要根据盒子的 初始状态 进行计算。

示例 1：

输入：boxes = "110"
输出：[1,1,3]
解释：每个盒子对应的最小操作数如下：
1) 第 1 个盒子：将一个小球从第 2 个盒子移动到第 1 个盒子，需要 1 步操作。
2) 第 2 个盒子：将一个小球从第 1 个盒子移动到第 2 个盒子，需要 1 步操作。
3) 第 3 个盒子：将一个小球从第 1 个盒子移动到第 3 个盒子，需要 2 步操作。将一个小球从第 2 个盒子移动到第 3 个盒子，需要 1 步操作。共计 3 步操作。

示例 2：

输入：boxes = "001011"
输出：[11,8,5,4,3,4]

提示：

  • n == boxes.length
  • 1 <= n <= 2000
  • boxes[i] 为 '0' 或 '1'

思路一：

计算出所有的球往右移至一个盒子和往左移至一个盒子的所有值，并将两者相加。

即：数组 left 保存了将下标 i 的所有左边的球移至 i 中所需要的操作数。
数组 right 保存了将下标 i 的所有右边的球移至 i 盒子中所需要的操作数。

返回数组则为 ： answer = left + right;

代码：

class Solution {
public:
    vector<int> minOperations(string boxes) {
        int n = boxes.size();
        vector<int> left(n,0),right(n,0),answer(n,0);
        for (int i = 1,steps = 0;i < n;i++){
            if (boxes[i - 1] == '1') steps++; // steps 为步数。
            left[i] = left[i - 1] + steps;
        }
        for (int i = n - 2,steps = 0;i >= 0;i--){
            if (boxes[i + 1] == '1') steps++;
            right[i] = right[i + 1] + steps;
        }
        for (int i = 0;i < n;i++) answer[i] = right[i] + left[i];
        return answer;
    }
};

复杂度分析：

  • 时间复杂度：O(n)。三次循环。
  • 空间复杂度：O(n)。两个数组。

思路二：

我们先将所有球移至第一个盒，计算出操作数。

移至下个盒子中的操作数就可以看成：在该盒子左边的球都需要往右移一步。右边盒子的球需要倒退一步。这样就只需要了解每个盒子的左边球的个数，以及右边盒子的个数。

代码：

class Solution {
public:
    vector<int> minOperations(string boxes) {
        int left = boxes[0] - '0', right = 0, operations = 0;
        int n = boxes.size();
        for (int i = 1; i < n; i++) {
            if (boxes[i] == '1') {
                right++;
                operations += i;
            }
        }
        vector<int> answer(n);
        answer[0] = operations;
        for (int i = 1; i < n; i++) {
            operations += left - right;
            answer[i] = operations;
            if (boxes[i] == '1') {
                left++;
                right--;
            }
        }
        return answer;
    }
};

复杂度分析：

  • 时间复杂度：O(n)。两次次循环。
  • 空间复杂度：O(1)。常数个数的变量。

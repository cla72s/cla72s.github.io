开始C++

发表于 2022-11-09 更新于 2022-11-13 分类于 C++

现在开始 c++ 的学习，因为发现 c 库中的函数太少，自己又比较的懒，自己不愿封装函数，又看到他人的代码那么简洁，所以开始了 C++ 学习。接下来粗略的记录一些 c++ 与 c 不同的地方。

进入C++

C 语言的所有语句都能放在 C++ 中运行。C++ 对 C 语言中的很多语句进行了补充。

main 函数：

在 C 语言中每个 main 函数都要在结尾添加 return 0; 语句以便给操作系统检查，该程序是否运行出错。由于程序员要在 C 中每个 main 函数都要填加 return 0; 给许多的程序员造成了一定的烦恼。所以 C++ 编译器让程序员可以不用输入 return 0; 在程序运行时自动添加上 return 0; 语句。

输入输出语句：

C 中的输入输出是以函数的形式存在。C++ 增加了流的概念，将输入设备和输出设备看成一个流。

在 C++ 中：

cout << endl;   // endl 与 "\n" 等价。
cin >> x ;      // x 为变量。
//<< 和 >> 表示了设备流的流向

字符串读取：

在 C++ 中：

cin.getline();
cin.get();
//两者区别：
//getline() : 读取一行到换行符为止，然后将换行符丢弃。
//get() : 读取一行到换行符为止，然后将换行符保留在输入流中。

赋值：

在 C++ 中：

int x = 5;  //与下面的全都等价。
int x(5);
int x{5};
int x = {5};

引入头文件

C 中的头文件都是 <string.h> 有个 .h 结尾。在 C++ 中将这些头文件进行了改变。改成了前缀 c 。例如：<cstring>

auto 的重新定义：

C 语言中 auto 是一个关键字，在 C++ 中 auto 是一个自动类型推断功能。

如：

auto x = 0;         // x 的数据类型就会被指定为 int 
auto y = 0.0;       // y 的数据类型就会被指定为 double
auto z = 1.3e12L;   // z 的数据类型就会被指定为 long

强制转换：

将一个数据类型进行强制转换：

(typeName) value;    // C中的
typeName (value);    // C++ 中新增的

C++ 中的格式让读者看起来像是使用了一个函数。

推广：

static_cast<typeName> (value);

结构体：

在 C++ 中减少了 typedef 的使用。

代码：

struct guest {
    char name[20];
    float weight;
};
// 想要申请结构可以省略 struct 
guest g ;

C++ 中的结构体也满足赋值的赋值形式：

guest g {"caoxing",60};     //这样就是一一对于结构体里面的值

指针：

C++ 新添了 new 语句。可以替代 c 中的 malloc 。

int* x = (int*) malloc(sizeof(int)); 
//可以用下面语句代替。
int* x = new int;
//数组：
int* x = (int*) malloc(sizeof(int) * 10);   
//可以用下面语句代替
int* x = new int [10];

但是在 C++ 中使用的 new 分配内存一定要配对一个 delete ;

int* ps = new int; 
...
delete ps; // ps 还是可以使用，只是删除了 new 分配的内存。

C++ 中的 一些类

string 类：

string类 是一种字符串的替代品。

string str1;       //定义一个 str1 对象、以动态分配内存，不需要指明大小。
string str2 = "jaguar";
string str3 = "ocelot";
str1 = str2 + str3; // 可以用 + 拼接两个 string 对象
str2 += str3;       // 这样也是可以的

string类 的 I/O：

string str;         // 起初创建时，str 的长度为 0 
getline(cin,str);   // 从输入流中得到一个字符串。
cout << str.size(); // 输出 str 的长度

vector 类 和 array 类：

模板类 vector 和 array 是数组的替代品。

  • 模板类 vector ：

模板类 vector 和 string 类相似，都是动态分配内存。

vector<typeName> vt(n_elem);     
// vt 是一个 vector<typeName> 对象,储存的个数为 n_elem 
// 在插入 或者添加时会自动调整长度。
  • 模板类 array ：

它是 C++ 11 新增的一个类。相比于 vector 类，他是一个静态内存分配的类，vector 形成动态分配产生了代价，使得其效率比较低。所以 array 对象 与数组一样，长度固定，效率和数组一样，但更方便、更安全。

下面声明创建一个名为 arr 的 array 对象，它包含 n_elem 个类型为 typeName 的元素。

array<typeName, n_elem> arr;    // n_elem 不能为变量
// 也可以初始化变量
array<double, 4> ad = {1.2, 2.1, 3.43, 4.3};

数组、vector 对象 、 array 对象 的相同与不同：

  • 相同点： 都可以用标准的数组表示法来访问各个元素。

  • 不同点：

  1. array 对象和 数组都保存在 栈内存 中，vector 对象保存在 堆内存 或者 自由存储区 中。
  2. array 对象可以直接赋值给另一个对象，而对于数组只能逐元素赋值。

数组 、 vector 对象 和 array 对象 的 for 循环（类似于java 的 for - each 循环）：

double prices[5] = {4.99, 10.99, 6.87, 7.99, 8.49}
for (double x : prices){
    cout << x << endl ;
}

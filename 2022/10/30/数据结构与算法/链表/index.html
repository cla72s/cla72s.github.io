链表

发表于 2022-10-30 更新于 2022-11-12 分类于 数据结构与算法

链表（List）是是一种线性表数据结构。它用不连续的内存空间，来存储一组具有相同类型的数据。

链表的优缺点：

优点：

  • 是一种动态数据结构。
  • 易于插入和删除。
  • 内存利用率高。

缺点：

  • 储存密度低，比较浪费内存，因为储存元素时，需要额外存储指针。
  • 不支持随机访问。
  • 查找元素效率低。

理解指针或引用的含义

要学好链表，首先得学好指针。

指针的理解：

将某个变量赋值给指针，实际上就是将这个变量的地址赋值给指针，或者反过来说，指针中存储了这个变量的内存地址，指向了这个变量，通过指针就能找到这个变量。

简单的来说就是：

int *p = data;
//则有以下两条：
*p == data;
p == &data;

数据的查找、插入和删除

查找：

查找只能遍历整个链表。时间复杂度 O(n)。

插入：

插入节点为 p2 ，将插入在 p1 之后。

代码实现：

p2->next = p1->next;
p1->next = p2;

时间复杂度为 O(1) 。

删除：

只需将该节点的上一个节点的 next 值指向这个节点的 next 即完成节点的删除。

代码实现：

p->next = p->next->next;

这样就将p的下一个节点删除了（还要将删除的空间给 free 掉）。

时间复杂度为 O(1) 。

边界问题（设置哨兵）

在插入删除头节点这个边界问题时，和其他节点时的代码不一样，那就需要使用不同的代码进行描述。

为了解决这种问题，我们可以在单链表的第一个节点之前设置个节点（一般称为哨兵节点），不保存数据只指向链表的第一个节点，这样，所有的操作都可以用相同的代码进行实现。

链表的各种实现

定义结构体：

typedef struct list{
    int data;
    struct list *next;
}List;

申请一个节点:

{
    List* node = (List*)malloc (sizeof(List));
    node->data = x;
    node->next = NULL;
}

链表头插：

//对于头节点不保存元素的链表：
{
    head->next = node;
}
//对于头节点要保存元素的链表：
{
    node->next = head;
    head = node;
}

链表尾插：

{
    List* Head = head;
    //遍历至链尾
    while(Head->next != NULL){
        Head = Head->next;
    }
    Head->next = node;
}

链表头删：

//对于头节点不保存元素的链表：
{
    List* node = head->next;
    head->next = node->next;
    free(node);
}
//对于头节点要保存元素的链表：
{
    List* node = head;
    head = head->next;
    free(node);
}

链表尾删：

{
    List* Head = head;
    //遍历至链尾
    while(Head->next != NULL){
        Head = Head->next;
    }
    Head = NULL;
}

寻找节点：

//要寻找的节点值为 x ;
{
    List* node = NULL;
    List* Head = head;
    while (Head->next != NULL){
        if (Head->data == x){
            node = Head;
            break;
        }
        Head = Head->next;
    }
    return node;
}

在指定节点值 x 前插入节点：

//要插入的节点为 node;
{
    List* Head = head;
    while (Head->next != NULL){
        if (Head->next->data == x){
            node->next = Head->next;
            Head->next = node;
            break;
        }
        Head = Head->next;
    }
}

删除指定节点值 x 的前节点：

{
    List* frontNode = head;
    List* node = frontNode->next;
    while (node->next != NULL){
        if (node->next->data == x){
            frontNode->next = node->next;
            free(node);
            break;
        }
        frontNode = node;
        node = frontNode->next;
    }
}

寻找倒数第 k 个节点：

{
    List* ansNode = head;
    List* node = head;
    //用双指针，一个指针先跑 k 个节点，答案节点再与其一起跑
    for (int i = 0;i < k;i++){
        node = node->next;
    }
    while (node != NULL){
        node = node->next;
        ansNode =ansNode->next;
    }
    return ansNode;
}

链表的释放：

{
    List* node = head;
    while (head != NULL){
        head = head->next;
        free(node);
        node = head;
    }
}
